# Naming Conventions

## Directories
- `snake_case` or `kebab-case` for standard folders.
- `PascalCase` for extracted skills (e.g., `SRSAnalysis`, `SRSRewrite`).

## Files
- Python files: `snake_case.py`.
- Markdown documents: Prefix with type (`SRS_Template_*.md`).
- Rule files: `kebab-case.md`.

## Python Code
- Classes: `PascalCase`.
- Functions and variables: `snake_case`.
- Constants: `UPPER_SNAKE_CASE`.
